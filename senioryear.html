Data Wrangling

My fall semester of senior year I took a Data Wrangling class, which goes over key topics and skills for data wrangling based on programming with R. It has been my favorite class I have taken throughout college.

My project was looking into the correlation between a country's happiness index score and their population change. I used R Studio to write the code and find statistics. There is a link for the fnal report and a link to go to the GitHub repository with the code.

Introduction to Econometrics

My fall semester senior year I took Intro to Econometrics which taught me about economic data analysis. This course covered topics including elementary statistics, structure of economic data, linear regression models, least squares estimation, hypothesis testing, and causal inference.

Through this project, my group used multilinear regression and data visualization that we learned during the semester to answer the research question, is there a causal relationship between population increases and the economic damages of natural disasters.
